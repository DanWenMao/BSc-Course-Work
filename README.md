# BSc-Coursework

This repository contains a selection of (well... actually, almost all of) my coursework completed during my B.Sc. in Planetary Science at Nanjing University, China.

- Files/ contains reports, theses, presentation slides, and other course materials.
- Codes/ contains source code written for various courses and projects.
- The directory structure evolved under maximum entropy (that means totally in a mess). Any attempt to reorganize it would probably violate conservation laws.

One interesting thing: I started my undergraduate journey in 2023, back when LLMs hadn't yet become everyone's default coding partner. As a result, many of the earlier assignments were completed entirely without AI assistance. You could say they belong to **one of the last generations of undergraduate coursework crafted entirely by human hands** before the LLM era truly took off. 

Looking back, I think that's pretty cool—you might even be able to see my growth (and my struggles, my lost sleep, my roar and curse) over the years.
